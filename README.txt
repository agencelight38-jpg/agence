▶ DÉPLOIEMENT RAPIDE (VERCEL)
1. Créez un compte sur https://vercel.com
2. Créez un nouveau projet et importez ce dossier (Upload)
3. Deploy → votre site est en ligne
4. Dans Settings → Domains → ajoutez agencelight.com puis suivez les instructions DNS

▶ DÉPLOIEMENT RAPIDE (NETLIFY)
1. https://app.netlify.com → Add new site → Deploy manually
2. Glissez-déposez ce dossier ou le fichier ZIP
3. Dans Domain settings → Add custom domain → agencelight.com → Suivez les DNS
